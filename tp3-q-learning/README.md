# TP3 - Q-Learning em Grid 4x4

Notebook que treina um robo a navegar em um laboratorio 4x4 com lama e celulas toxicas usando Q-Learning com politica epsilon-greedy.

## Estrutura
- `notebooks/tp3.ipynb`: notebook principal com treino e visualizacao.
- `docs/tp3_material.pdf`: material original do trabalho (enunciado/relatorio).

## Como rodar
1. Crie um ambiente com Python 3.9+.
2. Instale dependencias: `pip install numpy pandas matplotlib`.
3. Abra `notebooks/tp3.ipynb` em Jupyter, VS Code ou Colab e execute as celulas.
4. Os hiperparametros (`alpha`, `gamma`, `epsilon`, episodios e passos maximos) podem ser ajustados nas primeiras celulas do notebook.
