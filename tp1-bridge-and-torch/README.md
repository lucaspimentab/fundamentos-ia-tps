# TP1 - Bridge and Torch Puzzle

Resolucao do puzzle da ponte e da tocha usando algoritmos de busca.

## Conteudo
- `entrega.py`: executa e compara DFS, BFS, UCS e A*.
- `a_star.py`, `bfs.py`, `dfs.py`, `ucs.py`: implementacoes dos algoritmos.
- `heuristic.py`: heuristica usada pelo A*.
- `utils.py`: geracao de sucessores, checagem de objetivo e impressao de passos.
- `docs/bridge_and_torch_puzzle.pdf`: PDF original do trabalho.

## Execucao rapida
Requer apenas Python 3.9+. Rode a partir da pasta `tp1-bridge-and-torch`:

```bash
python entrega.py
```

A saida mostra custo total, numero de nos expandidos e a sequencia de estados para cada algoritmo.
